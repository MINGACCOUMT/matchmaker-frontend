## 1. 登录页面 Login.vue 增强

参考 generated_11.png 设计：

### 布局
- 顶部品牌区：UkiUki Logo + "欢迎来到 UkiUki" + "遇见心动的你"
- 登录卡片（毛玻璃效果，大圆角）

### 功能
- **Tab 切换**："密码登录" / "短信登录" 两个 tab，选中状态底部有粉色下划线
- **密码登录模式**：
  - 邮箱/手机号输入框（带图标）
  - 密码输入框（带眼睛显示/隐藏切换）
  - "忘记密码？" 文字链接
- **短信登录模式**：
  - 邮箱/手机号输入框
  - 验证码输入框 + "获取验证码" 按钮（倒计时效果）
- **主按钮**：粉色渐变 "登录" 按钮
- **次级按钮**："注册新账号" 边框按钮
- **第三方登录**：微信、QQ、抖音图标按钮（圆形白底）
- **用户协议**：底部复选框 + "我已阅读并同意《用户协议》和《隐私政策》"

## 3. 首页/仪表盘 Dashboard/Home.vue 重构

参考 generated_12.png 设计，适配相亲场景：

### 布局
- **顶部导航**：Logo + 搜索框 + 通知钓钟 + 用户头像
- **三栏布局**：
  - 左栏：推荐用户列表
  - 中栏：匹配概览/发现卡片
  - 右栏：数据统计

### 左栏 — 推荐用户
- 标题"推荐给你" + 筛选下拉
- 用户列表项：头像、昵称、年龄、城市、匹配度进度条
- 每项有心动/不感兴趣操作
- "查看全部推荐 →"

### 中栏 — 匹配概览
- 标题"匹配概览" + 视图切换
- 匹配卡片：用户头像、昵称、简介、标签、匹配度环形进度
- 底部"查看详情" + "开始聊天"

### 右栏 — 数据统计
- 统计数字卡片（总推荐、已匹配、待处理、新消息）
- 匹配趋势折线图
- 用户分布环形图

## 4. 设置页面 Settings.vue 新增

参考 generated_13.png 设计：

### 布局
- iOS 风格分组列表
- 顶部导航栏：返回 + "设置" + 搜索

### 功能模块
1. **个人设置**
   - 用户信息卡片（头像、昵称、ID）
   - 个人资料
   - 修改头像
   - 主题色（当前粉色圆点）
2. **账号管理**
   - 账号与安全
   - 修改密码
   - 绑定手机
   - 绑定邮箱
3. **隐私设置**
   - 隐私权限管理
   - 隐藏在线状态（Toggle）
   - 谁可以看我的资料
   - 黑名单管理
4. **通知设置**
   - 消息通知（Toggle）
   - 声音提醒（Toggle）
   - 震动提醒（Toggle）
   - 邮件通知（Toggle）
5. **系统设置**
   - 深色模式
   - 语言与地区

### 视觉风格
- 每个设置项：左侧图标 + 标题 + 右侧内容/Toggle/箭头
- Toggle 开关使用粉色主题（开启粉色，关闭灰色）
- 分组标题使用粉色系

## 技术要求
- Vue 3 + Tailwind CSS
- 使用已有的全局样式系统（main.css 中的 .card, .btn-primary, .input, .tag 等）
- 所有新增/修改的页面需要响应式支持移动端
- 保持粉色渐变背景和毛玻璃卡片风格
- 添加飘浮装饰元素（小星星、心形等 CSS 动画）
